# Inject-JavaScript-via-PDFrw-example

- Запуск с параметрами по умолчанию:
`pdf_injector.py path/to/orig.pdf`

- Все параметры:
`        pdf_injector.py path/to/orig.pdf
        -n      ScriptName
        -o      path/to/out.pdf
        ( -S      path/to/script | 
        -s      someJSScript() )
        -i      [OpenAction | Annots | Names]`
